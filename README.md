# Senior Developer Problem Solving Approach

This Python script demonstrates the difference in problem-solving approaches between mid-level and senior developers. It highlights how seniors analyze root causes and propose holistic solutions, while also mentoring others.

## Language

`python`

## How to Run

Save the code as `senior_developer_problem_solving.py` and run it from your terminal using `python senior_developer_problem_solving.py`.
